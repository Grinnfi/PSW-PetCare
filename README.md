

 # 🐾 PetCare - Sistema de PetShop

Este projeto foi desenvolvido como trabalho acadêmico e consiste em um **website de petshop**, onde os usuários podem:

* 🛒 Comprar produtos para seus pets
* 📅 Realizar agendamentos de serviços (banho, tosa, etc.)

O sistema foi pensado para facilitar a experiência do cliente, centralizando compras e serviços em uma única plataforma.


## 🚀 Tecnologias utilizadas

* HTML, CSS, JavaScript
* React


## 📦 Como executar o projeto localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

### 1. Clonar o repositório

Abra o terminal e execute:

```bash
git clone https://github.com/Grinnfi/PSW-PetCare.git
```



### 2. Acessar a pasta do projeto

```bash
cd PWS-PetCare
```


### 3. Instalar as dependências

```bash
npm install
```

ou

```bash
yarn install
```


### 4. Executar o projeto

```bash
npm start
```

ou

```bash
npm run dev
```


### 5. Acessar no navegador

Após iniciar o servidor, acesse:

```
http://localhost:3000
```

*(ou a porta indicada no terminal)*


## 📌 Funcionalidades

* Cadastro de usuários
* Listagem de produtos
* Compra de produtos
* Agendamento de serviços
* Interface amigável
